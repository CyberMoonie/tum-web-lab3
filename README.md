# Shadow Depths — Landing Page (Lab 3)

Responsive landing page for **Shadow Depths**, a fictional metroidvania game.

## What's New in Lab 3

- **Bootstrap 5 migration** — layout, navbar, grid, and utilities replaced custom CSS
- **Responsive design** — three breakpoints (992px, 768px, 480px) adjust all elements
- **Mobile-only elements** — CTA banner and community section visible only on small screens (`d-md-none`)
- **Animated mascot** — SVG shadow-wisp that appears after 2 s in the bottom-right corner, bounces/tilts continuously, and shows a welcome message on hover

## Tech Stack

- HTML5, CSS3
- Bootstrap 5.3
- SVG + CSS Animations
